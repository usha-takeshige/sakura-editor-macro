# サクラエディタ マークダウンマクロ集

サクラエディタでマークダウン編集を快適にするためのマクロ集です。

## 概要

このリポジトリには、マークダウンファイル編集時によく使う操作を効率化するためのマクロが含まれています。テキスト整形、インデント操作、リスト自動継続など、マークダウン編集を快適にする機能を提供します。

## インストール

1. このリポジトリをクローンまたはダウンロードします
2. `.js` ファイルを任意のフォルダに配置します（例：`C:\sakura_macros\`）
3. サクラエディタで各マクロにキーバインドを設定します

### キーバインドの設定方法

1. サクラエディタのメニューから「設定」→「共通設定」を開く
2. 「マクロ」タブを選択
3. 各マクロファイルを登録し、対応するキーバインドを設定

## 機能

### テキスト整形

| 操作 | ショートカット | マクロファイル | 説明 |
|------|----------------|----------------|------|
| **太字** | `Ctrl+B` | `markdown_bold.js` | 選択範囲を `**` で囲む（選択がない場合は `****` を挿入し、カーソルを中央に配置） |
| **斜体** | `Ctrl+I` | `markdown_italic.js` | 選択範囲を `*` で囲む（選択がない場合は `**` を挿入し、カーソルを中央に配置） |
| **引用符** | `Ctrl+2` | `markdown_quote.js` | 選択範囲を `"` で囲む |
| **インラインコード** | ``Ctrl+` `` | `markdown_code.js` | 選択範囲を `` ` `` で囲む |

### インデント操作

| 操作 | ショートカット | マクロファイル | 説明 |
|------|----------------|----------------|------|
| **インデント増加** | `Ctrl+]` または `Tab` | `markdown_indent_increase.js` | 現在行のインデントを2スペース増やす |
| **インデント減少** | `Ctrl+[` | `markdown_indent_decrease.js` | 現在行のインデントを2スペース減らす |

### マークダウン補助機能

| 操作 | ショートカット | マクロファイル | 説明 |
|------|----------------|----------------|------|
| **自動継続** | `Enter` | `markdown_enter.js` | リスト、番号付きリスト、引用の自動継続 |

#### リスト自動継続

リスト行（`-`, `*`, `+` で始まる行）で `Enter` を押すと、次の行も自動的にリストとして継続します。

```markdown
- アイテム1 [Enter]
- （自動で挿入される）
```

空のリスト行で `Enter` を押すと、リストマーカーが削除されます。

#### 番号付きリスト自動インクリメント

番号付きリスト行（`1.`, `2.` など）で `Enter` を押すと、次の番号が自動的にインクリメントされます。

```markdown
1. 最初のアイテム [Enter]
2. （自動で挿入される）
```

空の番号付きリスト行で `Enter` を押すと、リストマーカーが削除されます。

#### 引用自動継続

引用行（`>` で始まる行）で `Enter` を押すと、次の行も自動的に引用として継続します。

```markdown
> 引用テキスト [Enter]
> （自動で挿入される）
```

空の引用行で `Enter` を押すと、引用マーカーが削除されます。

## 使用例

### テキストの太字化

```markdown
これは通常のテキストです
```

「通常の」を選択して `Ctrl+B` を押すと：

```markdown
これは**通常の**テキストです
```

### リストの作成

```markdown
- タスク1 [Enter]
- （カーソル位置）
```

自動的にリストマーカーが挿入されるので、続けて入力できます。

### インデントの調整

```markdown
- 親アイテム
  - 子アイテム（Ctrl+] でインデント）
```

## マクロファイル一覧

- `markdown_bold.js` - 太字（`**text**`）
- `markdown_italic.js` - 斜体（`*text*`）
- `markdown_quote.js` - 引用符（`"text"`）
- `markdown_code.js` - インラインコード（`` `text` ``）
- `markdown_indent_increase.js` - インデント増加
- `markdown_indent_decrease.js` - インデント減少
- `markdown_enter.js` - リスト・引用の自動継続
